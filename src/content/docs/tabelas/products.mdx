Armazena o catálogo de produtos de cada empresa. O campo `stock` é **calculado pelo banco** via trigger — nunca atualizar diretamente.

## Schema

<TableSpec fields={[
  { name: "id", type: "uuid", nullable: false, description: "PK gerado automaticamente" },
  { name: "company_id", type: "uuid", nullable: false, description: "Empresa dona", references: "companies.id" },
  { name: "name", type: "text", nullable: false, description: "Nome do produto" },
  { name: "sku", type: "text", nullable: true, description: "Código de referência (único por empresa)" },
  { name: "stock", type: "numeric", nullable: false, description: "Saldo atual — atualizado pelo trigger trg_apply_stock_movement" },
  { name: "unit", type: "text", nullable: false, description: "Unidade de medida (ex.: un, kg, cx)" },
  { name: "min_stock", type: "numeric", nullable: true, description: "Estoque mínimo para alerta" },
  { name: "created_at", type: "timestamptz", nullable: false, description: "Timestamp de criação" },
  { name: "updated_at", type: "timestamptz", nullable: false, description: "Timestamp de última alteração" }
]} />

## Índices

| Índice | Coluna(s) | Por quê |
|--------|-----------|---------|
| PK | `id` | Busca por ID |
| UNIQUE | `(company_id, sku)` | SKU único por empresa |
| `products_company_idx` | `company_id` | Listagem por empresa |
